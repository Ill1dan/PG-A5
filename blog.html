Question-1: What are the different ways to select an element in the DOM?

We can select an elements in DOM in the following ways:

  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:

innerHTML is a property used to fetch the HTML content of the element, including any HTML tags inside the element. We can also set the HTML elements using innerHTML.

innerText:

innerText is a property used to fetch the text content of the element, excluding any HTML tags inside the element. We can also change the text content of the element using innerText.

textContent:

textContent is a property used to fetch the text content of the element, including any hidden text inside the element but excluding any HTML tags inside the element. We can also change the text content of the element using textContent.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in which we add a single event listener to a parent element that will handle the event for all its child, whether they are present now or added in the future. It allows us to avoid adding event listeners to specific element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in which the event is first captured by the target element and then propagated to the parent elements in DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating Elements:

We can create elements using the createElement() method.

Adding Elements:

We can add elements to the DOM using the appendChild() method.

Removing Elements:

We can remove elements from the DOM using the removeChild() method.
